# دليل البدء السريع

## 🚀 خطوات سريعة للبدء

### 1. تثبيت الحزم
```bash
npm install
```

### 2. تشغيل المشروع
```bash
npm run dev
```

### 3. فتح المتصفح
افتح `http://localhost:3000`

## ⚙️ التخصيص السريع

### تغيير رقم الواتساب
`lib/whatsapp.ts` → غيّر `WHATSAPP_PHONE`

### تغيير روابط السوشيال
`lib/config.ts` → عدّل `instagram` و `email`

### تغيير الألوان
`tailwind.config.ts` → عدّل `accent.DEFAULT`

### إضافة منتج جديد
`data/products.ts` → أضف عنصر جديد في المصفوفة

## 📸 إضافة الصور

ضع الصور في: `public/images/`

استخدم المسار: `/images/filename.jpg`

## 📝 ملاحظات

- تأكد من أن الصور موجودة قبل إضافة المنتج
- رقم الواتساب بدون رموز (`966501234567`)
- جميع المسارات تبدأ من `public/`
